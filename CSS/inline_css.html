This is example of inline CSS

This page has example of inline css, inline css is written inside the html tag. inline css has the highest priority in term of css styling. even there is css in the external file or head section, inline css will override the external css or head css property.

Head section css is also known as internal CSS, as the css styling is written inside the HTML file
