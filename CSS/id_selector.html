Title Heading

This page has the example of CSS using ID selectors, to demonstrate how to apply unique styles to specific elements. the way to use ID selectors is to add an `id` attribute to the HTML element and then target it with the `#` symbol in the CSS. ID selectors are unique and can only be used once per page, it can be used to style individual elements uniquely.

ID selectors have higher priority than class selectors and element selectors.

Styling using ID selectors can be written in external CSS files.
